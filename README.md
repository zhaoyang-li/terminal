# Build Setup
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8081
npm run dev

# build for production with minification
npm run build -- prod
(dist/production)

# build for rc with minification
npm run build -- rc
(dist/rc)

# build for dev with minification
npm run build -- test
(dist/test)

```

# Browser support
Modern browsers and IE 10+.
